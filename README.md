# 🌍 Life Expectancy Prediction App

A professional Machine Learning web application that predicts Life Expectancy using healthcare, socio-economic, and mortality indicators.

## 🚀 Technologies Used

- Python
- Scikit-Learn
- Streamlit
- Plotly
- Seaborn
- Matplotlib
- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet
- Decision Tree
- Random Forest
- KNN
- SVR
- AdaBoost
- XGBoost
- CatBoost
- LightGBM
- Gradient Boosting
- Voting Regressor
- Stacking Regressor
- Bagging Regressor

## 📊 Features

- Interactive Professional UI
- Feature Importance Visualization
- Prediction Analytics Dashboard
- Ensemble Learning Models
- Hyperparameter Tuning
- Glassmorphism Design

## 🏆 Final Model

Random Forest Regressor

### Model Performance

- Test R² Score: 96.79%
- RMSE: 1.67
